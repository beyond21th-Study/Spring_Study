## 스터디 진행 방식

> **읽기, 쓰기, 발표하기, 토론하기**
> 
- `읽기`: 해당 주차의 학습 범위 읽어오기
- `쓰기`: 학습 범위와 관련있는 나만의 주제를 선정하여 깃허브 리드미 작성하기
- `발표하기`: 미팅 시간에 공부한 내용에 대해 발표하기 (랜덤 한명)
- `토론하기`: 일요일 자정까지 학습 범위와 관련있는 토론 주제를 깃허브에 작성하고 월요일 미팅 전까지 다른 멤버의 공부 자료에 대해 조사해와서 미팅 시간에 토론하기
